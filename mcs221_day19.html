🛒 1. Association Rule Mining (ARM) & Apriori Algorithm

Market Basket Analysis • Support • Confidence • Frequent Itemsets

📖 Explain

Association Rule Mining discovers hidden patterns/relationships between items in transactional databases. Rule format: X → Y (e.g., {Diapers} → {Beer}).

  • Support(X→Y): P(X ∪ Y) = transactions containing both / total transactions
  • Confidence(X→Y): P(Y|X) = transactions with both / transactions with X
🛍️ Example: 100 transactions: 20 contain Diapers & Beer → Support = 20%
If 30 contain Diapers → Confidence = 20/30 = 66.7%
WEKA Implementation: Associate → Apriori → set minSupport, minConfidence

📝 2. Text Mining & TF-IDF (Vector Space Model)

Term Frequency • Inverse Document Frequency • Document Vectors

📖 Explain

TF-IDF (Term Frequency-Inverse Document Frequency) measures term importance in a document corpus.

  • TF(t,d): Frequency of term t in document d
  • IDF(t): log(Total Documents / Documents containing t)
  • TF-IDF = TF × IDF — high value = important term for that document
📄 Example: 1000 reviews: "cinematography" appears 4 times in 1 review, 0 times in others → High TF-IDF!
"the" appears 50 times but in all reviews → TF-IDF ≈ 0
Formula: IDF(t) = log(N / df(t)) | TF-IDF(t,d) = tf(t,d) × idf(t)

📝 Exam-Style Questions & Answers (MCS-221 PYQ Pattern)

Q1. Explain Association Rule Mining (ARM). Also, explain the two phases involved in the ARM process with the help of an example. (Direct PYQ - 10 Marks)
Answer: ARM discovers hidden relationships between items in transactional databases. Two Phases: 1. Frequent Itemset Generation: Find all itemsets with support ≥ minSupport (e.g., {Bread, Milk} appears in 40% of transactions). 2. Rule Generation: Generate rules with confidence ≥ minConfidence (e.g., {Bread} → {Milk} with 75% confidence). Used in market basket analysis for cross-selling insights.
Q2. Define 'Support' and 'Confidence' in the context of Association Rules.
Answer: Support(X→Y): Proportion of transactions containing both X and Y (frequency). Confidence(X→Y): Conditional probability that Y is purchased when X is purchased (reliability). Support = |X∪Y|/N, Confidence = |X∪Y|/|X|.
Q3. Discuss vector space modeling for representing text documents. With reference to this modeling, explain TF-IDF and Inverse Document Frequency (IDF). (Direct PYQ - 10 Marks)
Answer: Vector Space Model represents documents as vectors of term weights. TF-IDF: TF measures term frequency in a document; IDF measures global rarity across corpus. TF-IDF = TF × IDF. High values indicate terms that are frequent in a specific document but rare overall, capturing document-specific keywords. IDF = log(N / df), where N = total docs, df = document frequency.
Q4. What is Web Mining? List its three main categories.
Answer: Web Mining applies data mining techniques to extract knowledge from web data. Three categories: Web Content Mining (text/images from pages), Web Structure Mining (hyperlink analysis), Web Usage Mining (user clickstream logs).
Q5. How can Web Usage Mining benefit an e-commerce website?
Answer: Web Usage Mining analyzes user navigation patterns from server logs. Benefits: 1. Personalized recommendations based on browsing behavior. 2. Identify cart abandonment points. 3. Optimize site navigation. 4. Detect fraudulent activities. 5. Improve user experience through A/B testing insights.
